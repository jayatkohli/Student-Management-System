# Student Management System (C++)

A simple console-based Student Management System built using C++.

## Features

- Add Student
- View All Students
- Search Student by Roll Number
- Update Student Details
- Delete Student
- Calculate Average Marks
- Display Class Topper
- Menu-Driven Interface

## Concepts Used

- Functions
- Arrays
- Loops
- Conditional Statements
- Strings
- User Input/Output

## Technologies

- C++
- Visual Studio Code
- GCC / MinGW

## How to Run

Compile:

```bash
g++ main.cpp -o student
```

Run:

```bash
./student
```

(On Windows)

```bash
student.exe
```
